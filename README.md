# Phonebook

A basic phonebook app which does the following:

Parses and displays JSON data from a URL.
Displays data in a responsive Bootstrap 4 table.
Allows users to search via the search bar.
Allows users to show number of entries.
Allows users to arrange (A-Z) by column.
Allows users to navigate the phonebook using pagination.

Future updates: V2 will allow users to Create, Update and Delete entries.
